# Neural Circuit Viewer

Public GitHub Pages export for the interactive neural circuit viewer.

## Publish

1. Push this repo to GitHub as a public repository.
2. In GitHub, open `Settings -> Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select `main` and `/(root)`.
5. Save.

After Pages finishes deploying, the viewer will be available from the repo's GitHub Pages URL.
